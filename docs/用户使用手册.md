# 毕业论文管理系统 - 用户使用手册

## 系统概述

毕业论文管理系统是一个基于 **Spring Boot + Vue 3** 的全栈 Web 应用，用于管理学生毕业论文的提交、审核与评阅流程。

---

## 环境要求

| 组件 | 版本要求 |
|------|----------|
| JDK | 17+ |
| Node.js | 18+ |
| Maven | 3.8+ |
| Docker | 20.10+ |
| Docker Compose | 2.0+ |

---

## 快速启动

### 启动所有服务（开发模式）

在项目根目录打开 PowerShell，执行：

```powershell
.\start-dev.ps1
```

启动后可访问：

| 服务 | 地址 |
|------|------|
| **前端界面** | http://localhost:3000 |
| **后端 API** | http://localhost:8080 |
| **MySQL** | localhost:13306 |
| **Redis** | localhost:6379 |

### 停止所有服务

```powershell
.\stop-dev.ps1
```

## 常见问题

### Q: 启动后访问 http://localhost:3000 提示 403 错误

**原因**: 可能是旧的 Node 进程占用端口或代理配置问题。

**解决方案**:

1. 执行 `./stop-dev.ps1` 停止所有服务
2. 重新执行 `./start-dev.ps1`

### Q: 后端启动失败，提示数据库连接错误

**解决方案**:

1. 确认 Docker 正在运行
2. 执行 `docker-compose -f docker-compose.dev.yml up -d`
3. 等待 10 秒后重试

### Q: 登录提示"密码错误"

**解决方案**:

- 确认使用正确的账号密码
- 查看 `sql/` 目录下的初始化脚本确认密码
- 或通过注册功能创建新账号
